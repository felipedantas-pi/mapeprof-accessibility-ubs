# Protocolo AE2SFCA - Teresina
Produto Técnico Tecnológico desenvolvido no âmbito do Mestrado Profissional (MAPEPROF/IFPI).

## Objetivo
Automatizar o cálculo de acessibilidade espacial a serviços de saúde utilizando o método **Areal Enhanced 2-Step Floating Catchment Area (AE2SFCA)**.

## Como usar
1. Instale as dependências: `pip install -r requirements.txt`
2. Coloque os dados brutos na pasta `data/raw`.
3. Execute os notebooks na ordem numérica (01 a 04).
